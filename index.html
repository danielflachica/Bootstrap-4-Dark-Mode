Bootstrap 4 Dark Mode Documentation

This mobile-friendly Dark Mode vs. Light Mode toggler was made entirely with Bootstrap 4, a couple of custom styles and one simple JS function. To see the effect, simply click on the button below. To use it in your own projects, you need to make sure your text-level elements make use of Bootstrap's text{-light|-dark|-white|-muted} class and your containers/wrappers/background elements use Bootstrap's bg{-light|-white|-dark} or the bg-black class that ships with this package.

Dark Mode CSS

The only custom CSS needed is shown below:

Copy! 
.bg-black {
  background-color: #1f1f1f;
}

Feel free to customize the hexadecimal value of the bg-black class' background-color property to your liking.

Dark Mode JavaScript

The only JavaScript needed (in addition to Bootstrap's core JS) for the Side Drawer is documented below. Be sure to include it before the end of the body tag.

Copy!

  ...
}
